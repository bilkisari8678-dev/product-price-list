# Product Price List

Website sederhana untuk menampilkan daftar produk dan harga, dengan fitur:

- Tambah pesanan
- Hitung total item dan harga
- Reset pesanan

## Cara Menjalankan

1. Clone repository ini.
2. Buka `index.html` di browser.
3. Atau deploy sebagai GitHub Pages:
   - Pilih repository → Settings → Pages → pilih branch `main` → `/root` → Save.
   - Buka URL yang muncul, misal: `https://username.github.io/product-price-list/`
